# 🛍️ Onlayn Savdo (E-Commerce Platform)

Zamonaviy va tezkor frontend texnologiyalari asosida yaratilgan Elektron Tijorat (E-Commerce) platformasi. Loyihada foydalanuvchi tajribasi (UI/UX), komponentlar arxitekturasi va toza kod (Clean Code) qoidalariga alohida e'tibor qaratilgan.

## 🚀 Texnologiyalar va Instrumentlar

* **React.js (Vite)** — Tezkor va modulli frontend infratuzilmasi.
* **Tailwind CSS** — Zamonaviy va moslashuvchan (responsive) stilizatsiya.
* **React Hooks** — Komponentlar holatini (state) samarali boshqarish.
* **Custom Hooks** — Sahifa sarlavhalari va takrorlanuvchi mantiqni optimallashtirish (`usePageTitle`).
